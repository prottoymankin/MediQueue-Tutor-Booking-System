# MediQueue – Tutor Booking System

[🚀 Live Demo](https://mediqueue-frontend-swart.vercel.app/)

## 📝 Description
MediQueue is a tutor booking web application that allows students to easily register, log in, and find tutors based on their needs. Users can search tutors by name and filter them by available dates, making it simple to discover the right tutor quickly.

The platform focuses on making tutor selection easy and efficient by removing manual scheduling problems. It provides a clean interface where students can view tutor profiles, check availability, and choose suitable tutors without confusion or delay.

## 🛠️ Technologies Used
### Frontend
- React.js
- NEXT.js
- Tailwind CSS 
- Hero UI

### Backend
- Express.js
- Mongodb
- JWT (Authentication)

### Authentication
- BetterAuth
- JWT

### Utilities/Libraries
- React-Hot-Toast
- React Icons

## 🚀 Core Features
- User authentication system for secure login and registration
- Search tutors by name with case-insensitive filtering
- Filter tutors based on available date range
- Dedicated tutor details page with full information
- Tutor booking system for scheduling learning sessions
- Add new tutors to the system with complete details
- Cancel booked sessions when needed
